# Wick & Glow

Welcome to Wick & Glow - Your Sanctuary for Holistic Well-being! This is the codebase for our premium aromatherapy e-commerce platform.

## About Wick & Glow

Wick & Glow is a premium aromatherapy e-commerce platform offering a curated selection of essential oils, diffusers, and wellness products. Our mission is to bring tranquility and holistic well-being to our customers through high-quality aromatherapy products.

## Features

- Premium aromatherapy products with variants and custom pricing
- User accounts with synchronized shopping cart
- Seamless guest checkout experience
- Cash on Delivery payment option
- Order history tracking (for both registered users and guests)
- Dynamic page builder with custom blocks
- Static generation with on-demand revalidation
- Live preview functionality
- Dark mode support
- Responsive design optimized for all devices
- SEO optimized for better visibility

## Technology Stack

- Next.js 15 for the frontend
- Payload CMS for content management
- TypeScript for type safety
- TailwindCSS for styling
- MongoDB for database
- Shadcn UI components

## Getting Started

1. Clone this repository
2. Copy `.env.example` to `.env` and configure your environment variables
3. Install dependencies:
   ```bash
   pnpm install
   ```
4. Start the development server:
   ```bash
   pnpm dev
   ```

## Environment Variables

Create a `.env` file in the root directory with the following variables:

```bash
COMPANY_NAME="Wick & Glow"
TWITTER_CREATOR="@srjaykikani"
TWITTER_SITE="https://srjay.com"
SITE_NAME="Wick & Glow"
PAYLOAD_PUBLIC_SERVER_URL=http://localhost:3000
NEXT_PUBLIC_SERVER_URL=http://localhost:3000
DATABASE_URI=your_mongodb_uri
PAYLOAD_SECRET=your_secret_key
```
